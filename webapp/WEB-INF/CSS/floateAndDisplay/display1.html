1.块级元素和行内元素

(1)h1-h6,div,p, 等都是块级元素；独占一行
(2) span,行内元素，不独占一行；
行内元素可以被包含在块级元素中，反之不行；
display： block 块元素； inline 行内元素，不独占一行；inlin-block 是行内块元素，可以内联，在一行；
float：设置成行内元素之后，可以设置float可以实现浮动，向左向右等；
注意使用float容易出现塌陷溢出等情况，参考2.解决；
clear:right,右侧不允许有浮动元素；left，左侧不允许有浮动元素；both，两侧不允许有浮动元素；none；
